# Optional HugoBlox Site Overrides

This folder contains the Hugo layout overrides used alongside the ORCID importer work.

## Included files

- `layouts/publications/single.html`
- `layouts/_partials/page_metadata_authors_apa.html`
- `layouts/_partials/functions/publication_apa.html`
- `layouts/_partials/functions/publication_details.html`
- `layouts/_partials/views/citation.html`

## Use

Copy the `layouts/` subtree into your HugoBlox site if you want:

- APA-style author formatting
- a citation block on publication pages
- publication metadata rendered more explicitly

These overrides are optional and independent from the importer itself.
